Welcome to the Data Processing App

Leveraging Python, Pandas, and GitHub Actions for efficient data workflows.

Project Overview

This application demonstrates a robust data processing pipeline. It takes an Excel spreadsheet, data.xlsx, converts it to CSV format, and processes it using a Python script, execute.py. The final processed data is then output as result.json, which is published via GitHub Pages.

Key Components

  • execute.py: The core Python script, fixed for Python 3.11+ and Pandas 2.3, handling data transformation.
  • data.xlsx: The initial data source in Excel format, converted to data.csv for processing.
  • data.csv: The CSV version of the data, derived from data.xlsx, used by execute.py.
  • result.json: The output of the processing, automatically generated and deployed via GitHub Actions.
  • GitHub Actions Workflow (.github/workflows/ci.yml): Automates linting with Ruff, script execution, and deployment to GitHub Pages.
